  - нові сценарії запуску

## 🚀 Швидкий старт

### Варіант 1 — автоматично
Запуск без запитів, з авто-вибором параметрів:

```bat
run-auto.bat
```

### Варіант 2 — вручну
Інтерактивний режим із підказками та вибором:

```bat
run-manual.bat
```

## ⚙️ Параметри запуску PowerShell-скрипта

`ukrainizator.ps1` підтримує такі параметри:

| Параметр | Опис |
|---|---|
| `-Silent` | Пропускає інтерактивні запити |
| `-Mode All` | Застосувати зміни для всіх користувачів |
| `-Mode Current` | Застосувати зміни лише для поточного користувача |
| `-NoRebootPrompt` | Не питати про перезавантаження — перезапустити автоматично |

### Приклади
```powershell
powershell -ExecutionPolicy Bypass -File .\ukrainizator.ps1
```

```powershell
powershell -ExecutionPolicy Bypass -File .\ukrainizator.ps1 -Silent -Mode All
```

```powershell
powershell -ExecutionPolicy Bypass -File .\ukrainizator.ps1 -Silent -Mode Current -NoRebootPrompt
```

## 🧠 Що робить скрипт

- встановлює український мовний пакет `uk-UA`
- перемикає мову інтерфейсу Windows на українську
- виставляє регіональні стандарти України
- налаштовує розкладки клавіатури: **UKR + ENG**
- прибирає російську мову та зайві розкладки
- створює точку відновлення системи
- показує покроковий прогрес, статуси та лог
- працює з підвищенням прав адміністратора

## 🛠️ Як це працює

Під час запуску скрипт:

1. перевіряє права адміністратора
2. перевіряє версію PowerShell
3. перевіряє доступ до інтернету
4. попереджає про системні зміни
5. створює точку відновлення
6. визначає режим встановлення
7. підтягує потрібні модулі
8. встановлює `uk-UA`
9. змінює мову інтерфейсу
10. застосовує регіональні стандарти України
11. очищує російські розкладки та мови
12. виконує фінальні налаштування й перезапускає Explorer
13. пропонує перезавантаження

## ❓ FAQ

### Чи можна запускати без запитів?
Так. Використай `run-auto.bat` або передай `-Silent -Mode All`.

### Чи змінює скрипт реєстр?
Так, але лише для потрібних налаштувань Windows і без небезпечних “оптимізацій” клавіатури.

### Чи створюється точка відновлення?
Так, скрипт намагається створити точку відновлення системи на початку виконання.

### Чому потрібне перезавантаження?
Бо зміни мови, регіону та розкладок застосовуються не повністю без restart.
