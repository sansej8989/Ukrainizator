# 🇺🇦 Ukrainizator v1.5.1

**Робимо Windows рідною, щоб комп'ютер нарешті заговорив по-людськи!** 🚀

Набридло, що твій комп'ютер розмовляє англійською, як турист у Львові, або взагалі мовою, яку ти не хочеш бачити? **Ukrainizator** — це твій цифровий перекладач, який одним кліком перетворює твою Windows на справжній український патріот-ПК.

## 🔥 Що він вміє?
- **Встановлює українську мову** (Language Pack) — щоб усе було зрозуміло.
- **Змінює мову інтерфейсу** (UI) — щоб кнопки "Пуск" та "Панель керування" не потребували словника.
- **Налаштовує розкладку клавіатури** — українська + англійська (US), щоб ти міг швидко перемикатися між написанням коду та повідомленням "Привіт!".
- **Перевіряє все за тебе** — права адміна, інтернет та версію PowerShell.

## 🛠 Як користуватися (Інструкція для лінивих)
1. **Завантаж** цей репозиторій.
2. **Двічі клікни** по файлу `run.bat`.
3. **Погодься** на все, що він запитає (тільки будь уважним, він попередить про перезавантаження).
4. **Перезавантаж комп'ютер** (або дозволь скрипту зробити це за тебе).
5. **Насолоджуйся** рідною мовою! 🇺🇦

## ⚠️ Важливі застереження (Читати обов'язково!)
- **Адмін-права:** Скрипту потрібні права адміністратора, бо він лізе в "мізки" системи.
- **Інтернет:** Без нього мовний пакет не завантажиться.
- **Точка відновлення:** Ми впевнені в себе, але Windows — штука примхлива. Створи точку відновлення системи перед запуском, якщо ти дуже обережний.
