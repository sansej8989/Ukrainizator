**Перетвори свій Windows на рідну українську мову за один клік!** 🚀  
Набридли англійські меню та чужі інтерфейси? **Ukrainizator** — твій персональний асистент для українізації Windows. Встановлює мовний пакет, змінює UI, налаштовує клавіатуру та звуково сигналізує про прогрес — все автоматично та без головного болю.

## 🔥 Основні можливості

- 🇺🇦 **Мовний пакет uk-UA** — повна локалізація системи.
- 💻 **Мова інтерфейсу** — меню, діалоги та повідомлення українською.
- ⌨️ **Розкладки клавіатури** — UKR + ENG, перемикання через **Shift+Alt**.
- ⚡ **Системна оптимізація** — Num Lock при старті, швидкий повтор клавіш, вимкнення залипання.
- 🌍 **Повна локалізація** — системна локаль та GeoID України.
- 🔤 **Гарантована кирилиця** — UTF-8 BOM, `chcp 65001`, шрифт Lucida Console (16pt).
- 🔊 **Звуковий супровід** — тональні сигнали, що наростають із кожним кроком + переможна мелодія.
- 📊 **Professional UI** — постійний список кроків з маркерами `[V]`, `[~]`, `[X]`, `[>>]`.
- 🔍 **Автоперевірка вимог** — адмін-права, інтернет, версія PowerShell.

## ❓ FAQ

**Чи потрібна переустановка після кожного оновлення Windows?**  
Так, великі оновлення можуть збити налаштування — просто запусти скрипт ще раз.

**Чи працює на Windows 10?**  
Так, скрипт сумісний з Windows 10 (1903+) та Windows 11.

**Чому на деяких ПК замість кирилиці були пусті місця?**  
Це була проблема кодування консолі. Починаючи з v3.0.0, скрипт автоматично встановлює UTF-8 (`chcp 65001`), правильний шрифт (`Lucida Console`) та зберігається з BOM-маркером — проблема повністю вирішена.

**Чи змінює скрипт реєстр?**  
Так, він використовує офіційні API Windows та мінімальні записи реєстру для налаштувань мови та шрифту.
