**Перетвори свій Windows на рідну українську мову за один клік!** 🚀  
Набридло англійські меню та чужі інтерфейси? **Ukrainizator** — твій персональний асистент для Ukrainizacji Windows. Встановлює мовний пакет, змінює UI та налаштовує клавіатуру — все автоматично та без головного болю.

## ✨ Чому саме Ukrainizator?
| ⚡ Швидкість | 🛡️ Безпека | 💫 Простота |
|--------------|-----------|-------------|
| Один скрипт — вся робота | Перевірка адмін-прав та інтернету | Крок-за-кроком інструкція |
| Автоматичне перезавантаження (за бажанням) | Створення точки відновлення рекомендовано | Жодних складних команд |

## 🔥 Основні можливості
- 🇺🇦 **Встановлення української мовної пакети** — повна локалізація системи.
- 💻 **Зміна мови інтерфейсу (UI)** — меню, діалоги та повідомлення українською.
- ⌨️ **Розумне налаштування мов** — автоматичне очищення від зайвих мов, встановлення UKR + ENG та перемикання через Shift+Alt.
- ⚡ **Системна оптимізація** — Num Lock при старті, прискорення повтору клавіш, вимкнення залипання та системного Beep.
- 🌍 **Повна локалізація** — встановлення системної локалі та GeoID України.
- 🔍 **Автоматична перевірка вимог** — адмін-права, підключення до інтернету, версія PowerShell.
- 💾 **Рекомендація створення точки відновлення** — для повної безпеки.

## 🚀 Як користуватися (3 простих кроку)
1. **Завантаж** репозиторій (кнопка `Code` → `Download ZIP` або склонуй через Git).
2. **Запусти від адміністратора** — двічі клікни на `run.bat` або запусти через PowerShell як Admin.
3. **Слідуй підказкам** — дозволь зміни та перезавантаж комп'ютер (скрипт може зробить це автоматично).

> ⚠️ **Важливо:** Для роботи потрібні права адміністратора та стабільне інтернет-з'єднання.

## ❓ Поширені питання (FAQ)
**Чи потрібна переустановка після кожного оновлення Windows?**  
Так, великі оновлення можуть збити мовні налаштування — просто запусти скрипт ще раз.

**Чи працює на Windows 10?**  
Так, скрипт сумісний з Windows 10 та 11.

**Чи змінює скрипт реєстр?**  
Так, він змінює параметри мови через офіційні API Windows.
