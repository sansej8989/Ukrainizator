Описание:
Собрано из официального образа.
— Superlite эта версия уже настроена - для опытных пользователей - подходит для игр и стриминга.
— Compact это нетронутая версия, без изменений, только удаление приложений/ненужных программ - для обычных пользователей - подходит для работы в офисе, ноутбуков, планшетов, серверов и т. д.

Версии при установке:
1. WINDOWS 11 PRO + COMPACT x64 — НЕ урезанная windows, только удаление приложений/ненужных программ - для обычных пользователей - подходит для работы в офисе, ноутбуков, планшетов, серверов и т. д.
2. WINDOWS 11 PRO + COMPACT + DEF x64 — То же самое, что в пункте 1. Но включен Windows Defender.
3. WINDOWS 11 PRO + SUPERLITE x64 — Эта версия уже настроена - для опытных пользователей - подходит для игр и стриминга.
4. WINDOWS 11 PRO + SUPERLITE + DEF x64 — То же самое, что в пункте 3. Но включен Windows Defender.
5. WINDOWS 11 PRO + SUPERLITE SE x64 — То же самое, что в пункте 3. Но добавлено меню "Пуск" как в Windows 7.
6. WINDOWS 11 PRO + SUPERLITE SE + DEF x64 — То же самое, что в пункте 5. Но включен Windows Defender.
*За Пуск в версии SUPERLITE SE отвечает программа StartAllBack

Язык:
Вставьте в PowerShell или в CMD по очереди:
Код: выделить все
powershell.exe Install-Language -CopyToSettings -ExcludeFeatures ru-ru

Код: выделить все
powershell.exe Set-SystemPreferredUILanguage -Language ru-ru

Код: выделить все
powershell.exe Restart-Computer

Интернет конечно обязателен.
p.s. можно заменить ru-ru на свой язык
